A flexible serializer for C++ objects, primitives, STL containers, and user-\/defined types. 

The {\ttfamily serializer} emits a structured, human-\/readable output inspired by formats like JSON or YAML, but uses custom token syntax and compile-\/time traits to determine how to serialize different types.
